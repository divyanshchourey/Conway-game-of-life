Conway's Game of Life

What is it?

Conway's Game of Life is a cellular automaton devised by mathematician John Horton Conway in 1970. It's a zero-player game, meaning its evolution is determined by its initial state, requiring no further input.

Rules:

  • Any live cell with fewer than two live neighbors dies (underpopulation)
  • Any live cell with two or three live neighbors lives on
  • Any live cell with more than three live neighbors dies (overpopulation)
  • Any dead cell with exactly three live neighbors becomes a live cell (reproduction)

How to play:

Click on cells to toggle them between alive and dead states. Press Start to begin the simulation. You can adjust the speed using the slider. Reset will clear the grid, and Randomize will create a random pattern.

Keyboard Shortcuts

  • Spacebar: Start/Stop simulation
  • R: Randomize the grid
  • C: Clear/Reset the grid
  • +: Increase number of cells (decrease cell size)
  • -: Decrease number of cells (increase cell size)
  • Esc: Return to main menu
  • T: Toggle between light and dark theme
